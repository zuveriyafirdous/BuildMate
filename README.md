# 🏗️ BuildMate — India's Trusted Construction Marketplace

A two-sided marketplace connecting homeowners with verified construction builders.

## 🌐 Live Site
👉 https://zuveriyafirdous.github.io/buildmate

## ✨ Features
- Homeowner & Builder dual login portals
- Milestone-based escrow payment verification
- Photo proof submission before payment release
- Live bid negotiation chat
- AI Vaastu Predictor
- Land price tracker (Hyderabad)
- Partial scope projects (painting, plumbing, interior, etc.)

## 🚀 How to Run Locally
Just open `frontend/index.html` in any browser — no setup needed.
